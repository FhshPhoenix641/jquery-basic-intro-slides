## 什麼是 Jquery ？

- jQuery 是一個 JavaScript 函式庫 (Javascipt Library)
- jQuery 的精神： 「write less, do more」
- jQuery 對 [DOM (Document Object Model)](https://developer.mozilla.org/zh-TW/docs/Web/API/Document_Object_Model) 操作有進行優化
- jQuery 作為老牌 Library，泛用性非常廣。<br />目前全世界約莫 9 成的網站依舊在使用 jQuery ！

  【來源: [W3Techs](https://w3techs.com/technologies/history_overview/javascript_library/ms/y)】

## 為什麼用 jQuery？

- 如同前述原因，簡單，易用，易學。

- 如果沒有要寫很大的系統，<br />&emsp;&emsp;&emsp;用 jQuery 來寫網頁速度很快。

### 舉個例子 🌰

---

<CodeSurfer>

```js title="這是一段要針對 <h1 id='title'>...</h1> 的程式碼" subtitle="是不是挺囉嗦的？"
// 抓取 DOM
const title = document.getElementById("title");

// 設定 #title 的樣式
title.style = "font-size: 6rem;";

// 設定 #title 裡的文字
title.innerText = "Javascript 方法好囉嗦！";
```

```js title="但是我們改成 jQuery 的話呢？" subtitle="簡短好多！"
// 變數設定 + 樣式設定 + 文字設定
$("#title")
  .css("font-size", "6rem")
  .text("jQuery 的程式碼好短！");
```

</CodeSurfer>

## jQuery 的使用

\- 時間就是金錢，朋友！

"$" 是我們在使用 jQuery 時的起手式！

---

<CodeSurfer>

```js title="利用 jQuery 操作 DOM 修改其內部文字" subtitle="先打上 $ 符號"
$
```

```js title="利用 jQuery 操作 DOM 修改其內部文字" subtitle="呼叫指定的物件"
$("#object")
```

```js title="利用 jQuery 操作 DOM 修改其內部文字" subtitle="呼叫物件的方法 (jQuery 設定的)"
$("#object")
  .text("Hello World")
```

</CodeSurfer>
